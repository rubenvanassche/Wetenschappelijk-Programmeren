# Data Fitting - Oefening 5

## Requirements
- C++11 Compiler
- Cmake 3.8
- GSL 2.4

Het kan zijn dat oudere versies van deze requirements werken maar dit is hoe mijn setup op mijn computer is en waar de oefening kon worden gecompileerd en draaide.

## Compileren
```
cmake .
make
```

## Runnen
```
./main
```
